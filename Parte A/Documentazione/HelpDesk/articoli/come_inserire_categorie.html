Come inserire le categorie?

  • Creato: 03/03/2022

    • |

  • Ultima modifica: 03/03/2022
Questa guida è riferita esclusivamente al configuratore

Per inserire le categorie sarà sufficiente compilare un file xml da rinominare Categorie.xml.
La categoria radice dovrà essere univoca, dunque non ci devono essere due radici con lo stesso nome.
Inoltre l'albero sotto la radice dovrà essere anch'essa univoca, quindi nell'albero di una radice non ci devono essere doppioni.
Per inserire più categorie sarà sufficiente inserire un'altra struttura <categoria> all'interno della struttura <categorie>

Non è necessario inserire come campi lo Stato di conservazione e la Descrizione, in quanto sono inserite di default

Esempio 
                <categorie>
                  <categoria>
                  ...
                  ...
                  </categoria>
                  <categoria>
                  ...
                  ...
                  </categoria>
                  <categoria>
                  ...
                  ...
                  </categoria>
                </categorie>

Esempio di file xml: 

              <categorie>
                <categoria>
                  <nome>Elettronica</nome>
                  <descrizione>Elettronica di consumo</descrizione>
                  <campi>
                      <campo>
                          <nome>Marca</nome>
                          <tipo>PAROLA</tipo>
                          <obbligatorio>false</obbligatorio>
                      </campo>
                      <campo>
                          <nome>Modello</nome>
                          <tipo>PAROLA</tipo>
                          <obbligatorio>true</obbligatorio>
                      </campo>
                  </campi>
                  <nome_categoria_padre>null</nome_categoria_padre>
                </categoria>
                <categoria>
                    <nome>Telefonia</nome>
                    <descrizione>Telefoni</descrizione>
                    <campi></campi>
                    <nome_categoria_padre>Elettronica</nome_categoria_padre>
                </categoria>
                <categoria>
                    <nome>Smartphone</nome>
                    <descrizione>Smartphone di ultima generazione</descrizione>
                    <campi>
                        <campo>
                            <nome>Processore</nome>
                            <tipo>PAROLA</tipo>
                            <obbligatorio>false</obbligatorio>
                        </campo>
                        <campo>
                            <nome>Memoria</nome>
                            <tipo>INTERO</tipo>
                            <obbligatorio>true</obbligatorio>
                        </campo>
                        <campo>
                            <nome>SDcard</nome>
                            <tipo>VEROFALSO</tipo>
                            <obbligatorio>false</obbligatorio>
                        </campo>
                        <campo>
                            <nome>Dimensione schermo</nome>
                            <tipo>DECIMALE</tipo>
                            <obbligatorio>false</obbligatorio>
                        </campo>
                    </campi>
                    <nome_categoria_padre>Telefonia</nome_categoria_padre>
                </categoria>
              </categorie>
